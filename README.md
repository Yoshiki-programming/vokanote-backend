# vokanote - AI単語帳バックエンド

**vokanote** は、AIを使って「単語登録」から「発音の準備」までをすべて自動で行う、スマートな単語帳アプリのサーバーです。

## 🚀 このアプリでできること

単語を入力するだけで、AIがその単語にぴったりの例文などを自動で作成します。
- **AIが例文を作成**: Geminiが、その単語に合わせた自然な例文と翻訳を自動で作ります。
- **ネイティブの声を再生**: AIが作った例文を、本物の人間のような声（音声データ）に変換します。
- **自動でお掃除**: 使い終わった単語を消したり、アプリを辞めたりするときは、保存された音声データも一緒に自動で消去して、無駄なデータが残らないようにします。

## 🛠️ 工夫したポイント

### 1. お金と時間の節約 (`GenerateHandler`)
AIを使うには少しだけお金（通信料）がかかります。そのため、「同じ単語を何度もAIに作らせない」仕組みを作りました。

- **すでにある単語は使い回す**: 新しく作る前に「前に作ったかな？」と確認します。同じ単語があれば、AIは使わずに保存してあるデータをすぐに見せるので、待ち時間もゼロになります。



### 2. ゴミを残さない削除機能 (`DeleteHandler`)
データを消すときに、文章だけ消えて「音声ファイルだけがネット上に残ってしまう」というミスが起きないようにしました。

- **まとめてお掃除**: 1つのボタンで「文章」と「音声」の両方を順番にきれいに消します。
- **退会時も安心**: ユーザーがアプリを辞めるときは、その人が作ったすべてのデータを一気に消去する機能を持たせています。



---

## 🌟 開発者としてのこだわり

1. **「もったいない」をなくす**: AIを無駄使いしない設計で、アプリを動かすコストを抑えています。
2. **データの整理整頓**: 「消したはずなのにデータが残っている」という不具合が起きないよう、消去の順番にこだわりました。
3. **安全第一**: 他人が勝手にデータを書き換えられないよう、Googleの安全な認証システムを導入しています。
